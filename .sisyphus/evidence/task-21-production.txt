## GitHub Repository Setup (Manual Steps Required)
gh CLI was installed but requires interactive authentication.
To create and push:

1. Run: gh auth login
   - Select GitHub.com
   - Choose HTTPS protocol
   - Authenticate with browser

2. Run: gh repo create honeycombo --public --source=. --push --description "HoneyCombo - 저비용 기술 뉴스 큐레이션 사이트"
